# Football Coaching Box Line

The coaching box line is the line that separates the team bench area
from the coaching box. This line should be a different color than the
bench area and coaching box, which may be the same color

## Usage

``` r
football_coaching_box_line(coaching_box_line_length = 0, feature_thickness = 0)
```

## Arguments

- coaching_box_line_length:

  The length of the line forming the coaching box line

- feature_thickness:

  The thickness with which to draw the line

## Value

A data frame of the bounding box of the coaching box line
