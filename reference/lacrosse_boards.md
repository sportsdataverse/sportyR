# Lacrosse Boards

The boards are the wall around the outside of the field that constrain
the playing surface. The boards are either typically ovular in shape, or
not present if the field is outdoors

## Usage

``` r
lacrosse_boards(
  field_length = 0,
  field_width = 0,
  corner_radius = 0,
  boundary_thickness = 0
)
```

## Arguments

- field_length:

  The length of the field

- field_width:

  The width of the field

- corner_radius:

  The radius of the corners of the boards

- boundary_thickness:

  The thickness with which to draw the boards

## Value

A data frame of the bounding coordinates of the boards
