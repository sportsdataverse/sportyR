# Soccer Center Mark

The center mark is where kickoffs for each half, as well as following
any goal, are taken. The radius should be given to the outside of the
mark. This feature is located at midfield

## Usage

``` r
soccer_center_mark(feature_radius = 0)
```

## Arguments

- feature_radius:

  The radius of the center mark on the pitch

## Value

A data frame containing the bounding coordinates of the center mark
