# Volleyball End Line

The lines on the court that run the full width of the court are referred
to as the end lines, with the full width of the line being considered in
bounds

## Usage

``` r
volleyball_end_line(court_width = 0, line_thickness = 0)
```

## Arguments

- court_width:

  The width of the court, measured from the exterior edges of the
  sidelines

- line_thickness:

  The thickness of the end line

## Value

A data frame containing the bounding box of the end line
