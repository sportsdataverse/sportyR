# Football Try Mark

The try mark is the mark from which all tries start. This line is
located directly on the line `y = 0`. This line is not typically
considered an official yard line, which is why it is created
independently

## Usage

``` r
football_try_mark(try_mark_width = 0, feature_thickness = 0)
```

## Arguments

- try_mark_width:

  The width (in the y-direction) of the try mark

- feature_thickness:

  The thickness of the try mark

## Value

A data frame of the bounding box of the try mark
