# Curling Courtesy Line

The courtesy lines are where players stand during the delivery process
of each stone when the opposing team is throwing

## Usage

``` r
curling_courtesy_line(line_thickness = 0, line_length = 0)
```

## Arguments

- line_thickness:

  The thickness of the courtesy line

- line_length:

  The length of the courtesy line, from the side wall towards the centre
  line

## Value

A data frame containing the bounding box of the courtesy line
