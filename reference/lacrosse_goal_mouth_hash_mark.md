# Lacrosse Goal Mouth Hash Mark

The goal mouth may have a hash mark that extends towards midfield from
the goal line extended. This is that hash mark, and should be anchored
using its outer edge

## Usage

``` r
lacrosse_goal_mouth_hash_mark(
  goal_mouth_hash_mark_length = 0,
  line_thickness = 0
)
```

## Arguments

- goal_mouth_hash_mark_length:

  The length from the goal line extended towards the center of the field
  that the hash mark extends

- line_thickness:

  The thickness of the hash mark extending from the goal mouth

## Value

A data frame containing the bounding coordinates of the goal mouth's
hash mark
