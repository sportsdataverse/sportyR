# Football End Line

The end line is the line beyond the back of the endzone. Its interior
edge is considered out of bounds

## Usage

``` r
football_end_line(feature_thickness = 0, field_width = 0)
```

## Arguments

- feature_thickness:

  The thickness of the boundary lines

- field_width:

  The width of the field

## Value

A data frame of the bounding box of the end line
