# Create an "X" Shape

Create a set of `x` and `y` coordinates that form an "X"-like shape

## Usage

``` r
create_x_shape(bar_length, bar_width, rotation = 45)
```

## Arguments

- bar_length:

  The length of one of the bars that forms the "X" shape, from outer
  edge to outer edge

- bar_width:

  The width of one of the bars that forms the "X" shape, from outer edge
  to outer edge

- rotation:

  The angle of rotation (in degrees) that the resulting "X"-like shape
  should be rotated

## Value

A data frame containing the points needed to draw the specified "X"
