# Hockey Neutral Zone

The neutral zone is the middle "third" of the rink. This is the area
between the two zone (blue) lines. The center of the neutral zone should
lie along the line x = 0

## Usage

``` r
hockey_neutral_zone(rink_width = 0, feature_thickness = 0)
```

## Arguments

- rink_width:

  The width of the rink

- feature_thickness:

  The length of the neutral zone

## Value

A data frame containing the bounding coordinates of the neutral zone
