# Hockey Boards

The boards are the wall around the outside of the rink that constrain
the playing surface. The boards are typically ovular in shape

## Usage

``` r
hockey_boards(
  rink_length = 0,
  rink_width = 0,
  feature_radius = 0,
  feature_thickness = 0
)
```

## Arguments

- rink_length:

  The length of the rink

- rink_width:

  The width of the rink

- feature_radius:

  The radius of the corners of the boards

- feature_thickness:

  The thickness with which to draw the boards

## Value

A data frame of the bounding coordinates of the boards
