# Tennis Net

The net divides the court into two halves, and should run through the
line `x = 0` when viewing the court in TV view

## Usage

``` r
tennis_net(feature_thickness = 0, net_length = 0)
```

## Arguments

- feature_thickness:

  The thickness of the net

- net_length:

  The length of the net

## Value

A data frame containing the bounding coordinates of the net
