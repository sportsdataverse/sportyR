# Create Surface Plot Base

Create the base for the `ggplot2` instance with the correct theme
elements

## Usage

``` r
create_plot_base(plot_background = NULL)
```

## Arguments

- plot_background:

  A hexadecimal string representing the color to use for the plot's
  background. Default: `NULL`

## Value

A `ggplot2` instance onto which the features will be added
