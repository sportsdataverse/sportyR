# Tennis Baseline Center Mark

The center mark identifies the center point of the
[`tennis_baseline()`](https://sportyR.sportsdataverse.org/reference/tennis_baseline.md).
The line should extend towards the net

## Usage

``` r
tennis_center_mark(center_mark_length = 0, feature_thickness = 0)
```

## Arguments

- center_mark_length:

  The length of the center mark as measured from the back edge of the
  baseline

- feature_thickness:

  The thickness of the center mark

## Value

A data frame containing the bounding coordinates of the center mark
