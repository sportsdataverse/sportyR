# Hockey Defensive Zone

The defensive zone is the left "third" of the rink in TV view. This is
the area that a team defends when attacking from left to right

## Usage

``` r
hockey_defensive_zone(
  rink_length = 0,
  rink_width = 0,
  feature_radius = 0,
  nzone_length = 0
)
```

## Arguments

- rink_length:

  The length of the rink

- rink_width:

  The width of the rink

- feature_radius:

  The radius of the corners of the boards

- nzone_length:

  The length of the neutral zone

## Value

A data frame of the bounding coordinates of the defensive zone
