# Hockey Offensive Zone

The offensive zone is the right "third" of the rink in TV view. This is
the area that a team attacks to try to score a goal when attacking from
left to right

## Usage

``` r
hockey_offensive_zone(
  rink_length = 0,
  rink_width = 0,
  feature_radius = 0,
  nzone_length = 0
)
```

## Arguments

- rink_length:

  The length of the rink

- rink_width:

  The width of the rink

- feature_radius:

  The radius of the corners of the boards

- nzone_length:

  The length of the neutral zone

## Value

A data frame of the bounding coordinates of the offensive zone
