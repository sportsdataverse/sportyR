# Baseball Home Plate

Home plate. This is a pentagonal shape with its back tip located at the
origin of the coordinate system. The angled sides of home plate
intersect the baselines

## Usage

``` r
baseball_home_plate(home_plate_edge_length = 0)
```

## Arguments

- home_plate_edge_length:

  The length of a single edge of home plate

## Value

A data frame that contains the boundary of home plate
