# Tennis Doubles Alley

The doubles alley is the area between the singles and doubles sideline.
It should run the entire length of the court

## Usage

``` r
tennis_doubles_alley(court_length = 0, feature_thickness = 0)
```

## Arguments

- court_length:

  The length of the court

- feature_thickness:

  The thickness of the doubles alley (this is the distance between the
  singles and doubles sidelines)

## Value

A data frame containing the bounding coordinates of the doubles alley
