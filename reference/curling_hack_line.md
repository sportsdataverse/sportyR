# Curling Hack Line

The hack line connects the two footholds at each hack. It should be
anchored at the terminus of the centre line (see
[`curling_centre_line()`](https://sportyR.sportsdataverse.org/reference/curling_centre_line.md)
for more information)

## Usage

``` r
curling_hack_line(line_thickness = 0, hack_width = 0)
```

## Arguments

- line_thickness:

  The thickness of the hack line

- hack_width:

  The width of the hack, measured from the outside of one foothold to
  the outside of the other

## Value

A data frame containing the bounding box of the hack line
