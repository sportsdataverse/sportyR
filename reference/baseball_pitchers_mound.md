# Baseball Pitcher's Mound

The pitcher's mound. This is where the pitcher's plate is located, but
the pitcher's plate is not necessarily centered on the pitcher's mound

## Usage

``` r
baseball_pitchers_mound(pitchers_mound_radius = 0)
```

## Arguments

- pitchers_mound_radius:

  The radius of the pitcher's mound

## Value

A data frame of the pitcher's mound's bounding coordinates
