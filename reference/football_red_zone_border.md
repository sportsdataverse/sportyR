# Football Red Zone Border

The field border (see
[`football_field_border()`](https://sportyR.sportsdataverse.org/reference/football_field_border.md))
may have a different color along the red zone than it does along the
rest of the field. This is not always the case, but the feature is
provided for convenience

## Usage

``` r
football_red_zone_border(feature_thickness = 0)
```

## Arguments

- feature_thickness:

  The thickness of the field border

## Value

A data frame of the bounding box of the red zone border
