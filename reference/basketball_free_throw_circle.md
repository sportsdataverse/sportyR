# Basketball Free Throw Circle (Outline)

The outline of the free throw circle. The interior filling area is
created via
[`basketball_free_throw_circle_fill()`](https://sportyR.sportsdataverse.org/reference/basketball_free_throw_circle_fill.md)

## Usage

``` r
basketball_free_throw_circle(
  overhang = 0,
  free_throw_circle_radius = 0,
  line_thickness = 0
)
```

## Arguments

- overhang:

  The arc length of the free throw circle that hangs past the free throw
  line

- free_throw_circle_radius:

  The radius of the free throw circle

- line_thickness:

  The thickness of the free throw line

## Value

A data frame of the bounding coordinates of the free throw circle
