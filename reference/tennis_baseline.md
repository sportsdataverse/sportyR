# Tennis Baseline

The baseline is the line behind which a player will serve the ball. It
spans the entire width of the court, and its back edge denotes the
furthest boundary inside of which a ball can land and be considered in
play

## Usage

``` r
tennis_baseline(court_width = 0, feature_thickness = 0)
```

## Arguments

- court_width:

  The width of the court

- feature_thickness:

  The thickness of the baseline

## Value

A data frame containing the bounding coordinates of the baseline
