# Basketball Half Court

Each half court spans from the inner edge of the baseline to the center
of the division line, and serves as the base layer of the court plot

## Usage

``` r
basketball_half_court(court_length = 0, court_width = 0)
```

## Arguments

- court_length:

  The length of the court

- court_width:

  The width of the court

## Value

A data frame of the bounding box of half a basketball court
