# Create Diamond

Create a set of `x` and `y` coordinates that form a diamond. This
function is a wrapper on
[`create_rectangle()`](https://sportyR.sportsdataverse.org/reference/create_rectangle.md)
above

## Usage

``` r
create_diamond(height, width, center = c(0, 0))
```

## Arguments

- height:

  The vertical height of the diamond

- width:

  The horizontal width of the diamond

- center:

  The center coordinates of the diamond

## Value

A data frame containing the points needed to draw the specified diamond
