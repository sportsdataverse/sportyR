# Soccer Half Pitch

Half of the pitch is located on each side of the halfway line (see
[`soccer_halfway_line()`](https://sportyR.sportsdataverse.org/reference/soccer_halfway_line.md)
for more information)

## Usage

``` r
soccer_half_pitch(pitch_length = 0, pitch_width = 0)
```

## Arguments

- pitch_length:

  The length of the pitch

- pitch_width:

  The width of the pitch

## Value

A data frame containing the bounding coordinates of the half of the
pitch
