# Lacrosse Below-Goal Marking

On some fields, there is a below goal marking. These are circles that
should mirror each other, with their center points used as anchors

## Usage

``` r
lacrosse_below_goal_marking(below_goal_marking_radius = 0)
```

## Arguments

- below_goal_marking_radius:

  The radius of the below goal marking

## Value

A data frame containing the bounding coordinates of the below goal
marking
