# Soccer Touchline

The lines that run the full length of the pitch are called the
touchlines. In some cases, they may also be referred to as the
sidelines, as they comprise the sides of the pitch

## Usage

``` r
soccer_touchline(pitch_length = 0, feature_thickness = 0)
```

## Arguments

- pitch_length:

  The length of the pitch

- feature_thickness:

  The thickness of the touchline

## Value

A data frame containing the bounding coordinates of the touchline

## Details

The line thickness will be uniform for all features on the pitch
